Welcome to the Agent Smith API service. This API processes new signup information, researches business domains, and generates personalized email drafts.

API Status

The API is currently running and ready to accept requests.

Main Endpoint

POST /api/process-signup

Processes a new signup and generates a personalized email draft.

Headers:

Content-Type: application/json
X-API-Key: your-api-key-here

Request Body:

{
  "email": "john@acmecorp.com",
  "name": "John Smith"
}

Web Interface

Agent Smith provides a web interface for managing jobs and submitting new signups:

  • Dashboard - View and manage all jobs
  • New Signup - Submit a new signup for processing
